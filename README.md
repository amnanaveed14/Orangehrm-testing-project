## How to Run Tests

1. Install required dependencies:
   - **Python**: `pip install -r requirements.txt`  
   - **Java (Maven)**: `mvn install`  

2. Run tests:
   - **Python:**  
     ```bash
     pytest tests/
     ```
   - **Java (TestNG):**  
     ```bash
     mvn test
     ```

3. View reports in `reports/` folder.
